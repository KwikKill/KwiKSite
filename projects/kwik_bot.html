Kwik Bot

While I was watching IT videos on youtube during my free time, I had the idea to make a bot discord. First without any real purpose, then as a base for future projects. Kwik Bot was born (I swear I'm not megalomaniac).

This bot was quickly my biggest personal and self-taught project. I first developed it through trial and error with many tutorials, then I got comfortable and moved on to reading the Discord api documentation. Once the base was done, I put this bot on a vps and added features over time. Some fun like the "B" response when someone writes "AH" but others useful like a command to get my school schedule. My biggest improvement has been my League of Legends stats analysis commands detailed in a separate article.
